# Bootcamp de Data Analytics - Resumen de Evaluaciones


*Welcome to my evaluations repository! This repository reflects the work I carried out during my Data Analytics bootcamp. Is entirely written in Spanish, as it was the language in which I worked on it.*

¡Bienvenidos a mi repositorio de evaluaciones! Este repositorio refleja el trabajo que realicé durante mi bootcamp de Data Analytics.

## Descripción general

Este repositorio contiene una serie de evaluaciones realizadas a lo largo de mi bootcamp de Data Analytics. A continuación, se presenta un resumen de cada evaluación y su contenido:

## Descripción Global

Durante el bootcamp de Data Analytics, se abordaron una variedad de temas, incluyendo programación en SQL y Python, estadística, manejo de datos con librerías como NumPy y Pandas, visualización de datos con Matplotlib y Seaborn, Machine Learning supervisado, limpieza de datos, proceso ETL, y visualización de datos con Tableau. Cada evaluación se centró en aspectos específicos de estos temas.

## Librerías Utilizadas

A lo largo de las evaluaciones, se utilizaron varias librerías clave, incluyendo:

- `NumPy`: Para operaciones numéricas y manipulación de arrays.
- `Pandas`: Para la manipulación y análisis de datos tabulares.
- `Matplotlib` y `Seaborn`: Para la creación de gráficos y visualización de datos.
- `Scikit-learn`: Para la implementación de algoritmos de Machine Learning supervisado.
- `Tableau Public`: Para la creación de visualizaciones interactivas.

## Estructura del Proyecto

El repositorio está organizado en carpetas correspondientes a cada una de las evaluaciones:

### Evaluación del Módulo 1: Pyhton básico y SQL básico

En esta evaluación se cubrieron los fundamentos de SQL y Python, incluyendo consultas SQL avanzadas. La estructura de la carpeta es la siguiente:

- `modulo-1-sprint-1`: Contiene el código y los ejercicios relacionados con el primer sprint del módulo 1. 
- `modulo-1-sprint-2`: Contiene los ejercicios y notebooks relacionados con el segundo sprint del módulo 1. 

En general, lo aprendido en este módulo se resume de la siguiente manera:
- Python:
    - Fundamentos de Python: variables, booleanos y operadores numéricos y binarios.
    - Estructuras de datos: listas, diccionarios, tuplas y sets.
    - Control de flujo: condicionales, bucles (for y while), break y continue.
    - Funciones y clases.
    - Manipulación de archivos y cadenas de texto.
    - Conexión entre Python y SQL.
- SQL:
    - Modelo relacional y creación de tablas.
    - Modificación de tablas e inserción de datos.
    - Consultas básicas con SELECT, WHERE, IS NULL, NOT NULL, ORDER BY, DISTINCT, LIMIT, OFFSET, BETWEEN, IN y AS.
    - Consultas avanzadas con MIN, MAX, SUM, AVG y COUNT.
    - Consultas con GROUP BY, HAVING y CASE.
    - Consultas en múltiples tablas utilizando JOIN, UNION, INTERSECT, EXCEPT, subconsultas y CTEs.


### Evaluación del Módulo 2

La primera parte del Módulo 2 se centró en estadística básica, Pandas, NumPy, y visualización de datos. La estructura de la carpeta es la siguiente:

- `modulo-2-sprint-1`: Contiene los ejercicios y notebooks relacionados con el primer sprint del módulo 2. 
- `modulo-2-sprint-2`: Contiene los ejercicios y notebooks relacionados con el segundo sprint del módulo 2. 


En general, lo aprendido en este módulo se resume de la siguiente manera:
- Numpy:
    - Uso de arrays y operaciones numéricas.
    - Indexación y manipulación de datos.
    - Realización de operaciones estadísticas y matemáticas.
    - Generación de conjuntos y distribuciones aleatorias.

- Pandas:
    - Trabajo con Series y Dataframes.
    - Carga, guardado y manipulación de datos.
    - Aplicación de métodos de Pandas.
    - Filtrado y unión de datos.
    - Uso de Groupby y Apply.

- Visualización:
    - Creación y personalización de gráficos con Matplotlib.
    - Creación y personalización de gráficos con Seaborn.

- Estadística:
    - Conceptos básicos de estadística descriptiva.
    - Cálculo de cuartiles, tablas de frecuencias y sesgos.
    - Análisis de tablas de contingencia, correlación y intervalos de confianza.

- Análisis Exploratorio de Datos:
    - Identificación y tratamiento de valores nulos y outliers.
    - Realización de análisis exploratorio gráfico.

- Limpieza de Datos:
    - Limpieza de strings y valores nulos.
    - Tratamiento de outliers.
    - Uso de Sklearn para la limpieza de valores nulos.

- ETL (Extract, Transform, Load):
    - Extracción de datos mediante API's.
    - Transformación y limpieza de datos.
    - Inserción de datos en SQL.
    - Creación de clases y funciones para automatización de ETL.
    - Implementación de pipelines ETL.
